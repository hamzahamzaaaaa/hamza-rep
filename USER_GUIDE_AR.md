## 1. نظام القص الذكي للآيات

### 🎯 ما هو القص الذكي؟

نظام متقدم يسمح لك بقص آية معينة من السورة بدقة 100% بناءً على توقيتات ملف LRC.

### 🔧 كيفية العمل:

#### الخطوة 1: اختيار الآية
```
1. افتح السورة المطلوبة
2. اضغط مطولاً على الآية المراد قصها
3. اختر "قص الآية" من القائمة
```

#### الخطوة 2: المعالجة التلقائية
النظام يقوم تلقائياً بـ:
- ✅ قراءة ملف LRC للعثور على توقيت بداية الآية
- ✅ إيجاد توقيت بداية الآية التالية (نهاية الآية الحالية)
- ✅ حساب الفارق الزمني بدقة
- ✅ قص ملف الصوت باستخدام FFmpeg
- ✅ توليد ملف Mini-LRC جديد يبدأ من `[00:00.00]`

#### الخطوة 3: الحفظ
- ✅ حفظ المقطع الصوتي في: `Hamza_Medbouh/verse_clips/`
- ✅ حفظ ملف المزامنة في: `Hamza_Medbouh/verse_sync/`
- ✅ إضافة إلى قاعدة البيانات المحلية

### 📊 مثال عملي:

**الآية 255 من سورة البقرة (آية الكرسي):**

ملف LRC الأصلي:
```
[05:23.450] اللَّهُ لَا إِلَٰهَ إِلَّا هُوَ الْحَيُّ الْقَيُّومُ ﴿٢٥٥﴾
[05:35.120] لَا تَأْخُذُهُ سِنَةٌ وَلَا نَوْمٌ ﴿٢٥٦﴾
```

بعد القص - Mini-LRC الجديد:
```
[ti:البقرة - الآية 255]
[ar:حمزة مدبوح]
[length:11670]

[00:00.00]اللَّهُ لَا إِلَٰهَ إِلَّا هُوَ الْحَيُّ الْقَيُّومُ
[00:00.500]اللَّهُ
[00:01.200]لَا
[00:01.800]إِلَٰهَ
...
```

### ⚙️ التفعيل:

لتفعيل القص الفعلي بالصوت، تحتاج إلى:

1. إضافة مكتبة FFmpeg:
```bash
flutter pub add flutter_ffmpeg
```

2. الكود جاهز في: `lib/core/services/verse_clip_engine.dart`

## 2. الوضع الزجاجي الشامل

### 🎨 ما هو الوضع الزجاجي؟

تصميم عصري يستخدم تأثير الضباب (Blur) لإنشاء واجهة شفافة تبدو كالزجاج المعتم.

### 🌟 المكونات الزجاجية:

#### 1. GlassContainer - الحاوية الزجاجية
```dart
GlassContainer(
  blurX: 10.0,
  blurY: 10.0,
  opacity: 0.15,
  child: YourWidget(),
)
```

#### 2. GlassCard - البطاقة الزجاجية
```dart
GlassCard(
  blurIntensity: 15.0,
  child: Text('مرحباً بالعالم'),
)
```

#### 3. GlassButton - الزر الزجاجي
```dart
GlassButton(
  onPressed: () {},
  child: Text('اضغط هنا'),
)
```

#### 4. GlassListTile - عنصر القائمة
```dart
GlassListTile(
  leadingIcon: Icons.folder,
  title: 'تحميلاتي',
  subtitle: 'عرض جميع السور المحملة',
  onTap: () {},
)
```

#### 5. GradientGlassBackground - الخلفية المتدرجة
```dart
GradientGlassBackground(
  gradientColors: [
    Color(0xFF1A141F),
    Color(0xFF2D1B3D),
  ],
  child: YourContent(),
)
```

### 🎯 أماكن التطبيق:

✅ **القوائم المنبثقة**: عند الضغط على النقاط الثلاث  
✅ **صفحة المشغل**: خلفية المشغل الكامل  
✅ **الإعدادات**: جميع عناصر الإعدادات  
✅ **بطاقات السور**: كل سورة في بطاقة زجاجية  

### ⚡ نصائح الأداء:

للأجهزة القديمة، قلل من شدة الضباب:
```dart
blurX: 10.0  // بدلاً من 20.0
blurY: 10.0  // بدلاً من 20.0
```

## 3. إدارة التحميلات

### 📥 أيقونة التحميل (سحابة + سهم):

عند بدء التحميل، تظهر أيقونة مميزة تتكون من:
- 🌥️ أيقونة السحابة (Cloud)
- ⬇️ أيقونة السهم (Arrow)
- 📊 شريط التقدم الدائري
- ⚡ سرعة التحميل اللحظية

### 📊 معلومات التحميل المعروضة:

```
┌─────────────────────────────┐
│  ☁️⬇️  سورة البقرة          │
│  ████████░░░░░░  65.3%     │
│  ⚡ 2.5 MB/s  |  3:45 متبقي│
└─────────────────────────────┘
```

### ✅ بعد اكتمال التحميل:

1. **تختفي** أيقونة التحميل من الواجهة الرئيسية
2. **تظهر** علامة "مكتمل ✓" باللون الأخضر
3. **يظهر** حجم الملف (مثال: "15.2 MB")

### 📁 الوصول إلى التحميلات:

**الطريقة 1:** من القائمة الرئيسية
```
النقاط الثلاث → تحميلاتي
```

**الطريقة 2:** من شريط التنقل السفلي
```
تبويب "التحميلات" (الأيقونة: 📥)
```

## 4. آياتي المختارة

### 🎵 ما هو قسم "آياتي المختارة"؟

قسم خاص يحتوي على جميع الآيات التي قمت بقصها، مع:
- 🎤 ملف الصوت المقصوص
- 📝 نص الآية كامل
- 🎯 مزامنة LRC دقيقة
- 🔁 ميزة التكرار (Loop) للتحفيظ

### 📋 عرض الآيات:

```
┌──────────────────────────────────┐
│ ✂️ البقرة - الآية 255            │
│ اللَّهُ لَا إِلَٰهَ إِلَّا هُوَ   │
│ ⏱️ 00:11.670  |  🔁 تكرار       │
│ ▶️ تشغيل                         │
└──────────────────────────────────┘

┌──────────────────────────────────┐
│ ✂️ يس - الآية 12                 │
│ إِنَّا نَحْنُ نُحْيِي الْمَوْتَى │
│ ⏱️ 00:08.340  |  🔁 تكرار       │
│ ▶️ تشغيل                         │
└──────────────────────────────────┐
```

### 🔁 ميزة التكرار (Loop):

**للتحفيظ:**
1. اضغط على الآية لتشغيلها
2. اضغط على أيقونة التكرار (🔁)
3. ستعاد الآية تلقائياً بشكل مستمر

**أنواع التكرار:**
- 🔂 تكرار الآية الواحدة
- 🔁 تكرار جميع الآيات المختارة
- ▶️ تشغيل عادي (بدون تكرار)

### 🎯 المزامنة الذهبية:

عند تشغيل الآية المقصوصة:
- ✅ يظهر التظليل الذهبي فوق النص
- ✅ يتحرك بسلاسة بين الكلمات
- ✅ متزامن 100% مع الصوت (Mini-LRC)

## 6. الموجة الصوتية الديناميكية

### 🎵 ما هي الموجة الصوتية؟

تصوير مرئي لحركة الصوت يتكون من 32 شريط ذهبي يتحرك ديناميكياً مع ترددات صوت القارئ.

### 🌊 كيف تعمل:

**المعادلات الرياضية:**
```dart
// موجة أساسية
double base = sin((_controller.value * 2 * pi) + (index * 0.5));

// موجة ثانية للحركة الطبيعية
double wave = sin((_controller.value * 4 * pi) + (index * 0.2));

// مغلف على شكل منحنى جرسي
double envelope = sin(pi * index / (_barCount - 1));

// الارتفاع النهائي
paintHeight = (widget.height * 0.2) + 
             (widget.height * 0.8 * ((base + wave + 2) / 4) * envelope);
```

### 🎨 المظهر:

**عند التشغيل:**
```
    ▃ ▄ ▆ ▇ █ ▇ ▆ ▄ ▃
   ▂▃▅▇██████████▇▅▃▂
  ▁▂▄▆▇████████████▇▆▄▂▁
```

**عند الإيقاف:**
```
    ▁ ▁ ▁ ▁ ▁ ▁ ▁ ▁ ▁
```

### 📍 أماكن الظهور:

1. ✅ المشغل المصغر (Mini Player)
2. ✅ المشغل الكامل (Full Player)
3. ✅ صفحة التشغيل الحالية (Currently Page)
4. ✅ صفحة آياتي المختارة

### ⚡ التحكم:

- **تبدأ بالحركة**: عند الضغط على ▶️ تشغيل
- **تتوقف**: عند الضغط على ⏸️ إيقاف مؤقت
- **تختفي**: عند إيقاف المشغل تماماً

## 7. التظليل الذهبي الانسيابي

### ✨ ما هو التظليل الذهبي؟

تأثير مرئي يسلط الضوء على الكلمة أو الآية التي يتم قراءتها حالياً، مع انتقال سلس بين الكلمات.

### 🎯 أنواع التظليل:

#### 1. تظليل الآية الكاملة
```dart
// في dynamic_quran_engine.dart
Container(
  decoration: BoxDecoration(
    color: AppColors.gold.withOpacity(0.2),
    borderRadius: BorderRadius.circular(8),
  ),
  child: Text(verseText),
)
```

#### 2. تظليل الكلمة (Karaoke-Style)
```dart
// في mushaf_engine.dart
TextSpan(
  text: word,
  style: TextStyle(
    color: isCurrentWord ? AppColors.gold : Colors.white,
    fontWeight: isCurrentWord ? FontWeight.bold : FontWeight.normal,
  ),
)
```

### 🌊 الانتقال السلس:

**بدون انتقال (قفزات):**
```
كلمة1 → [قفزة] → كلمة2 → [قفزة] → كلمة3
```

**مع انتقال سلس (المُنفّذ):**
```
كلمة1 → 〜〜〜 → كلمة2 → 〜〜〜 → كلمة3
   (تدرج لوني متحرك)
```

### ⚙️ المعلمات:

```dart
AnimationController(
  vsync: this,
  duration: Duration(milliseconds: 300),  // مدة الانتقال
  curve: Curves.easeInOut,               // منحنى الحركة
);
```

### 🎨 الألوان:

- **اللون الأساسي**: ذهبي (`#FFD700`)
- **الشفافية**: 20% للخلفية، 100% للنص
- **التوهج**: `BoxShadow` بلون ذهبي خافت

## 8. الأسئلة الشائعة

### ❓ كيف أقص آية معينة؟

**الإجابة:**
1. افتح السورة المطلوبة
2. اضغط مطولاً على الآية
3. اختر "قص الآية" من القائمة
4. انتظر حتى يتم المعالجة (5-10 ثوانٍ)
5. ابحث عن الآية في "آياتي المختارة"

### ❓ أين تُحفظ الآيات المقصوصة؟

**الإجابة:**
```
المسار: /data/user/0/com.hamzamedbouh.quran/
├── Hamza_Medbouh/
    ├── verse_clips/          ← المقاطع الصوتية
    │   ├── surah_2_verse_255.mp3
    │   └── surah_36_verse_12.mp3
    └── verse_sync/           ← ملفات المزامنة
        ├── surah_2_verse_255.lrc
        ├── surah_2_verse_255.json
        └── surah_36_verse_12.lrc
```

### ❓ لماذا لا يعمل القص الفعلي؟

**الإجابة:**
تحتاج إلى تفعيل FFmpeg:
```bash
flutter pub add flutter_ffmpeg
```

ثم أعد بناء التطبيق:
```bash
flutter clean
flutter build apk --release
```

### ❓ كيف أقلل استهلاك البطارية؟

**الإجابة:**
1. قلل شدة الضباب في الوضع الزجاجي:
   - من 20.0 إلى 10.0
2. استخدم الموجة الصوتية البسيطة (4 أشرطة بدلاً من 32)
3. أغلق المشغل عند عدم الاستخدام

### ❓ كيف أحذف الآيات المقصوصة؟

**الإجابة:**
1. افتح "آياتي المختارة"
2. اضغط مطولاً على الآية
3. اختر "حذف"
4. سيتم حذف:
   - ✅ الملف الصوتي
   - ✅ ملف LRC
   - ✅ ملف JSON
   - ✅ السجل من قاعدة البيانات

### ❓ هل يمكن مشاركة الآيات المقصوصة؟

**الإجابة:**
نعم! كل آية مقصوصة تحتوي على:
- ✅ ملف صوتي مستقل
- ✅ ملف LRC للمزامنة
- ✅ ملف JSON للبيانات

يمكنك مشاركة الملفات الثلاثة مع أي شخص يستخدم التطبيق.

### ❓ كيف أكرر آية للتحفيظ؟

**الإجابة:**
1. افتح "آياتي المختارة"
2. اضغط على الآية لتشغيلها
3. اضغط على أيقونة التكرار (🔁)
4. ستتغير إلى (🔂) - تكرار آية واحدة
5. ستعاد الآية تلقائياً بشكل مستمر

لإيقاف التكرار:
- اضغط مرة أخرى على الأيقونة

### ❓ لماذا تظهر السور المحملة في قائمة التحميل؟

**الإجابة:**
بعد اكتمال التحميل:
- ✅ تختفي أيقونة التحميل (سحابة + سهم)
- ✅ تظهر علامة "مكتمل ✓" باللون الأخضر
- ✅ ينتقل السهم إلى قسم "تحميلاتي"

### ❓ كيف أجد سورة محملة معينة؟

**الإجابة:**
**الطريقة 1:**
```
التحميلات → شريط البحث → اكتب اسم السورة
```

**الطريقة 2:**
```
القائمة الرئيسية → النقاط الثلاث → تحميلاتي
```

### ❓ هل يمكن نقل التحميلات إلى SD Card؟

**الإجابة:**
نعم، لكن يحتاج إلى:
1. إضافة إذن `WRITE_EXTERNAL_STORAGE`
2. تغيير مسار التحميل في `download_provider.dart`

```dart
final String downloadPath = '/storage/emulated/0/Hamza_Medbouh/';
```
